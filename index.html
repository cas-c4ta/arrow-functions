Arrow Functions

«Arrow Functions sind eine verkürzte Schreibweise für Funktionen. ‘Arrow’ bezieht sich auf die Pfeilähnliche Form: =>

Herkömmliche Schreibweise


    function sum(a, b) { return a + b }
    sum(2, 3) // 5
    sum(8, 4) // 12

Verkürzte Schreibweise


    const sum = (a, b) => a + b
    sum(2, 3) // 5
    sum(8, 4) // 12
  • Bei nur einer Zeile können die geschweiften Klammern {} weggelassen werden.
  • Bei nur einer Zeile kann das return Statement weggelassen werden.
  • Gibt es nur ein Argument, kann auch die runde Klammer () weggelassen werden.

Anwendung

  • Nützlich ist dies, wenn Funktionen als Einschub in einer Zeile benötigt werden, die dann sehr lange wird.
  • Bsp. 1) Funktionen mit Callback
  • Bsp. 2) Array-Methoden wie forEach oder map

    const myArr = [
      { x: 10, y: 24 },
      { x: 72, y: 12 },
      { x: 39, y: 74 },
      { x: 41, y: 33 }
    ]

    myArr.forEach(pos => console.log(pos.x))
